Who Am I?

Hey. My name is Chris. Over the years I've worn a number of different hats: I've been a teacher; scuba divemaster; photographer; political advisor; traveller. Now I'm a web designer and developer.

I've lived on four continents, met a lot of people, and seen a lot of stuff. Now based in Brisbane, I've been transforming my skillset for the post-Covid world. I'm proficient in HTML, CSS, and Javascript, and I'm about to start working on further refining my backend skills.

When the world starts to open up again, you'll find me on an unknown beach somewhere working remotely.

Current Projects

I'm always trying to keep a few projects on the go.

Project 1: Australian Wildlife Word Game.

This work is an evolution of the Game Show App. It fetches data from the Australian Wildlife API, and randomly generates a phrase based on a common Australian animal. When the game is completed, an animal profile is generated to educate the user about the conservation status of the animal. Images are dynamically populated using both the iNaturalist and WikiMedia APIs.

Click here to view the site in action, or alternatively explore the source code in the GitHub repo.

Project 2: Ongoing portfolio redevelopment.

The initial design proposal for this portfolio rapidly evolved throughout the development of the project. Ultimately, what I feel looks aesthetically pleasing is bloated with some unnecessary design structures. Limitations have emerged through a fixed height section design. A mix of flexbox and absolute-positioned content was used to allow for text to overlap images - but in retrospect, this could be designed far more smoothly with CSS Grid. I will revisit this project.
